---------- Requisitos do projeto -----------

-Criação de conta:

    -Tela de criação de conta com opções para criar a conta diretamente no site ou através de plataformas como Steam, Epic Games e Google.
    - A tela de criação de conta terá duas caixas de texto para inserir o e-mail e a senha.
    - Abaixo das caixas de texto, haverá um botão para prosseguir com o preenchimento dos dados.
    - Também haverá 3 botões/ícones que permitirão a criação de conta através das plataformas mencionadas.
    - Após o cadastro, um e-mail de confirmação será enviado para a conta cadastrada. A confirmação por e-mail será necessária para acessar o site. (opção de cadastro através do site)

Login:

    -Tela de login com opções para fazer login utilizando a conta do site ou através de plataformas como Steam, Epic Games e Google.
    -A tela de login terá duas caixas de texto para inserir o usuário (e-mail ou nome de usuário) e a senha.
    -Haverá um botão "Lembrar de mim" e um link "Esqueci minha senha" que redirecionará para a tela de recuperação de senha.
    -Abaixo desses elementos, haverá um botão "LOGIN" para verificar as credenciais e acessar a plataforma.
    -Também haverá 3 botões/ícones que permitirão fazer login utilizando as plataformas mencionadas.

Recuperação de senha:

    -Na página de recuperação de senha, haverá uma caixa de texto para inserir o e-mail da conta cadastrada.
    -Abaixo da caixa de texto, haverá um botão para enviar a confirmação de recuperação de senha para o e-mail informado.
    -Após a confirmação do e-mail, o usuário será direcionado para a próxima tela onde poderá informar a nova senha.
    -Nessa tela, haverá duas caixas de texto para inserir e confirmar a nova senha.
    -Abaixo dessas caixas de texto, haverá um botão de confirmação que executará a troca de senha e direcionará para a página de login.

Perfil:

    -Visão pública:
        -O perfil terá uma foto de perfil, nome do usuário e um texto de descrição.
        -Haverá uma aba com os amigos do usuário, uma aba de configurações por jogo, uma seção para exibir o time ao qual o usuário pertence (caso exista) e os jogos que o usuário joga atualmente (escolhidos pelo usuário).
    
    -Visão privada:
        -Além das funcionalidades da visão pública, o perfil terá um botão para editar o perfil do usuário.

Configurações do usuário:

    -Visão pública:
        -Ao clicar na aba de configurações do usuário no perfil, será exibida uma lista de jogos nos quais o usuário cadastrou suas configurações em forma de cards.
        -Ao clicar em um card, o usuário será direcionado para uma página que exibirá todas as configurações utilizadas pelo usuário para o jogo selecionado.
    
    -Visão privada:
        -Além das funcionalidades da visão pública, o usuário terá a opção de editar suas configurações.
        -Haverá dois botões, um para editar as configurações de um jogo já cadastrado e outro para cadastrar configurações para um novo jogo.

Amigos:

    -Visão pública:
        -Ao clicar na aba de amigos, será exibida uma lista dos amigos do usuário.
        -Se o usuário clicar em um amigo da lista, será direcionado para o perfil do amigo respectivo.

    -Visão privada:
        -Além das funcionalidades da visão pública, o usuário poderá adicionar amigos a partir de um usuário específico ou a partir da lista de amigos de plataformas como Steam, Epic Games, etc.
        -Também haverá um botão para excluir amigos da lista.

Players profissionais:

    -Haverá uma aba com cards referentes a cada jogo.
    -Ao clicar no card de um jogo, será exibida uma lista dos perfis dos jogadores profissionais desse jogo.

Campeonatos:

    -Serão exibidas as datas dos campeonatos em andamento para cada jogo.
    -Também serão exibidos os próximos campeonatos para cada jogo.

Time:

    -Haverá uma lista de times e uma caixa de pesquisa para procurar um time.
    -Ao entrar na página de um time, será exibida a logo do time, uma descrição sobre o time, os campeonatos em que o time está participando e os integrantes do time.
    -Clicar na logo do time no perfil de um usuário direcionará para a página do time correspondente.

Social:

    -Fóruns de discussão:
        -Haverá uma lista de discussões criadas por outros usuários.
        -Haverá um botão para criar uma nova discussão.

    -Transmissões ao vivo:
        -Será exibida uma lista das principais transmissões ao vivo na Twitch para cada jogo.

Modo claro e escuro:

    -O site terá suporte aos modos claro e escuro, permitindo ao usuário escolher entre eles.
